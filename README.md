# Data Analysis Portfolio

A collection of applied data analysis projects completed as part of a Data Analyst career track, using Python for data cleaning, exploratory analysis, statistical testing, and visualization.

**Author:** Ujjwal Aditya Birenji
**Programme:** MSc Business Analytics, Aston University

---

## Projects

### 1. Crime Pattern Analysis — Los Angeles Crime Data
Analyzed LA crime records using Python (pandas) to identify peak crime hours, most affected age groups, and night-time crime concentration by area.
📁 `/la-crime-analysis`

### 2. Hypothesis Testing — Gender Comparison in International Soccer
Applied statistical hypothesis testing (Python, scipy) to compare scoring outcomes between men's and women's FIFA World Cup matches, determining whether observed differences were statistically significant.
📁 `/soccer-hypothesis-testing`

### 3. Investigating Netflix Movies
Explored Netflix's content catalog using Python (pandas) to uncover trends in movie duration, genre distribution, and release patterns over time.
📁 `/netflix-movies-analysis`

### 4. Exploring NYC Public School Test Scores
Analyzed NYC public school SAT performance data to identify top-performing schools and boroughs, and explored patterns in score distribution across the city.
📁 `/nyc-schools-analysis`

### 5. Visualizing the History of Nobel Prize Winners
Analyzed over a century of Nobel Prize data using Python (pandas, matplotlib) to explore trends in winner demographics, countries, and categories over time.
📁 `/nobel-prize-history`

### 6. Data Manipulation with Python
Applied core data manipulation techniques (filtering, grouping, merging, reshaping) using pandas to clean and transform raw datasets for analysis.
📁 `/data-manipulation-python`

---

## Tools Used

Python (pandas, numpy, matplotlib, scipy), Jupyter Notebooks

## Notes

Each project folder contains the notebook/script and a short summary of the approach and findings. These were completed as structured learning projects; datasets and starting frameworks are sourced from DataCamp's Data Analyst career track.
